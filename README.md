# Historical Buildings Facade Damage Assessor

ВКР по программе магистратуры "Искусственный интеллект" НИУ ВШЭ.

`Разработка метода оценки степени повреждения фасадов исторических
зданий на основе анализа изображений и пространственных зависимостей
между зданиями.`

## Описание

В работе предлагается метод оценки степени повреждения фасадов исторических зданий, сочетающий семантическую сегментацию дефектов на фотографиях
фасадов с анализом пространственных зависимостей между зданиями в масштабах 4 городских районов. На основе результатов сегментации формируются
количественные индексы повреждения фасада для различных типов повреждений. Городская историческая застройка представляется в виде графа, вершины которого
соответствуют отдельным зданиям, а рёбра - их пространственной и признаковой близости. Графовая нейронная сеть позволяет восстанавливать неизвестные значения индексов повреждения по известным значениям индексов соседних вершин и признакам самих зданий. Подход апробирован на примере исторической застройки
4 районов г. Санкт-Петербурга.

## Установка

Проект использует [uv](https://docs.astral.sh/uv/) для управления зависимостями.

```bash
# Клонировать репозиторий
git clone <repo-url>
cd historical-buildings-facade-damage-assessment

# Создать виртуальное окружение и установить зависимости
uv sync

# Активировать окружение
# Linux / macOS
source .venv/bin/activate
# Windows
.venv\Scripts\activate
```

### Установка torch-geometric

`uv sync` не устанавливает torch-geometric автоматически из-за зависимости от версии CUDA.
После `uv sync` выполните отдельно:

```bash
# для CPU
pip install torch-scatter torch-sparse -f https://data.pyg.org/whl/torch-2.2.0+cpu.html

# для CUDA 11.8
pip install torch-scatter torch-sparse -f https://data.pyg.org/whl/torch-2.2.0+cu118.html
```
